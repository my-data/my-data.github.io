My Data

M2M Monitoring and Control

About Company Page


My-Data specializes in real time control of critical processes. The low cost online SCADA packages use the the mobile phone network or ethernet to display analogue, digital and pulse counter data. Critical threshold can be set which trigger both text messages and emails to alert key personal.

Applications

  • Factory Automation
  • Process Control
  • Water Treatment
  • Agriculture
  • Transport
  • Security
